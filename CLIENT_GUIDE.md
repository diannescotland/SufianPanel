# Sufian Panel - Guide d'Installation

## Etape 1: Installer Docker Desktop

1. Telecharger Docker Desktop: https://www.docker.com/products/docker-desktop/
2. Lancer l'installateur et suivre les instructions
3. Redemarrer l'ordinateur si demande
4. Ouvrir Docker Desktop et attendre qu'il soit pret (icone verte en bas a gauche)

---

## Etape 2: Extraire l'Application

1. Extraire le fichier ZIP sur le **Bureau**
2. Vous aurez un dossier "Sufian Panel" sur votre Bureau

---

## Etape 3: Debloquer les Fichiers (Important!)

Windows bloque parfois les fichiers telecharges. Pour debloquer:

1. Clic droit sur le dossier **Sufian Panel**
2. Cliquer sur **Proprietes**
3. Si vous voyez "Ce fichier provient d'un autre ordinateur...", cocher **Debloquer**
4. Cliquer **OK**

---

## Etape 4: Lancer l'Application

1. Ouvrir le dossier **Sufian Panel**
2. Double-cliquer sur **start.bat**
3. Si Windows demande une confirmation, cliquer **Executer quand meme**

L'application va demarrer automatiquement. Attendre environ 1-2 minutes la premiere fois.

---

## Etape 5: Utiliser l'Application

Le navigateur s'ouvre automatiquement. Sinon, ouvrir:

**http://localhost:3000**

---

## Arreter l'Application

Double-cliquer sur **stop.bat**

---

## En Cas de Probleme

### "Docker n'est pas demarre"
1. Ouvrir Docker Desktop depuis le menu Demarrer
2. Attendre que l'icone devienne verte (en bas a gauche)
3. Relancer start.bat

### Docker Desktop ne demarre pas
- Verifier que la virtualisation est activee dans le BIOS
- Redemarrer l'ordinateur

### Page blanche ou erreur dans le navigateur
- Attendre 1-2 minutes, l'application est en train de demarrer
- Rafraichir la page (F5)

### "Windows a protege votre ordinateur"
- Cliquer sur **Informations complementaires**
- Cliquer sur **Executer quand meme**
